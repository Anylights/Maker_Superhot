Astroon v1.1.0

This package contains the Pencil .pen source and the fonts required for accurate text rendering.

Recommended workflow:
1. Keep every font file in the same directory as the .pen file.
2. Do not move the fonts into a nested fonts folder; Pencil only resolves sibling font files reliably.
3. If Pencil still shows fallback text, install the font files, then reopen the .pen file.
