## 五、风险项

| 风险 | 缓解策略 |
|------|---------|
| 分块渲染时屏幕外方块破坏状态丢失 | blockState_ 记录全局状态，节点只是视觉缓存 |
| 500m地图 AI 寻路性能 | scanPlatforms 限制为 ±25 层 |
| 动态 slot 分配颜色不连续 | 颜色 = slot % 4，不依赖连续性 |
| 爆炸同步给所有客户端的带宽 | 只发送给爆炸位置附近的客户端 |
